# Codes

1.感知：针对复杂动态环境人机交互感知资源利用率低的难题，提出了人体姿态实时交互感知跟踪方法，实现了对未知环境的动态感知和自主认知， 

2.调控：针对未知环境探索复杂度高、未知动态系统行为演化学习难、系统演化决策复杂等难题，提出了基于有限测量信息的复杂集群系统行为演化与自主决策方法，构建了混合知识驱动的动态系统演化学习与自主决策理论，有效提升了复杂系统自主决策的可靠性和稳定性，

3.调控：针对不可靠环境下调控抗扰性差等难题，提出了复杂不确定系统安全稳健调控方法，实现了对未知动态环境下复杂系统的智能学习与安全稳健调控，有效提升了复杂系统调控的稳定性和安全性，

4.调控：针对不可靠环境下调控抗扰性差等难题，提出了复杂不确定系统安全稳健调控方法，实现了对未知动态环境下复杂系统的智能学习与安全稳健调控，有效提升了复杂系统调控的稳定性和安全性，

5.决策：针对未知环境探索复杂度高等难题，提出了多重知识表达的分层探索方法，实现了非完全信息下复杂动态系统自主决策，

6.感知：针对复杂动态环境人机交互感知资源利用率低的难题，构建了多重知识与感知融合的未知环境认知学习理论与方法，有效提升了复杂系统在未知动态环境下的认知准确性和鲁棒性，

7.感知：针对复杂动态环境精准建图难等难题，提出了复杂环境自适应感知方法和跨模跨域大尺度融合建图方法，构建了多重知识与感知融合的未知环境认知学习理论与方法，实现了对未知环境的动态感知和自主认知，

8.建模：针对复杂动态系统耦合关联程度高，面向复杂任务约束完成程度低等问题，提出了复杂耦合关联集群的演化模型，形成了复杂耦合关联的动态系统建模理论与方法，为复杂动态系统的稳健调控与自主决策方法研究提供了理论基础。
